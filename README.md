# Тестовое задание от Grid Dynamics (done by Igor Bezmestin)

> A Vue.js project

## Запуск проекта

При помощи npm:

``` bash
# установить зависимости
npm i

# запуск сервера в режиме dev (разработки), сервер расположен по адресу localhost:8080
npm run dev

```

Либо при помощи yarn:

``` bash
# установить зависимости
yarn install

# запуск сервера в режиме dev (разработки), сервер расположен по адресу localhost:8080
yarn dev

```
