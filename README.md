# Тестовое задание от Grid Dynamics

> A Vue.js project

## Build Setup

При помощи npm:

``` bash
# установить зависимости
npm i

# запуск сервера в режиме dev (разработки), сервер расположен по адресу localhost:8080
npm run dev

# для компиляции сервера на продакшн
npm run build
```

Либо при помощи yarn:

``` bash
# установить зависимости
yarn install

# запуск сервера в режиме dev (разработки), сервер расположен по адресу localhost:8080
yarn dev

# для компиляции сервера на продакшн
yarn build
```
